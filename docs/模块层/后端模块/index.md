## 🎯 重点模块推荐

基于模块重要性和复杂度，建议优先了解：

1. **MCP服务器模块** - 核心协议实现和工具注册
2. **Web服务器模块** - Socket.IO集成和会话管理  
3. **服务器协调器** - 多服务器实例管理
4. **工具模块集合** - 基础设施和辅助功能
